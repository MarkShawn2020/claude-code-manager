## 🎯 核心信息

**产品名称**: Claude Code Manager (ccm)  
**最新版本**: v2.9.0  
**核心功能**: 增强 Claude Code 开发体验的集成工具  
**GitHub**: https://github.com/markshawn2020/claude-code-manager

## 🚀 最新更新：多状态栏支持

```bash
npm i -g claude-code-manager
ccm sl init              # 一键安装
ccm sl select <name>     # 切换风格
```

内置三种状态栏风格：
- **vibe-genius**: 全功能版
- **vibe-genius-wind**: 随机表情版
- **minimal**: 极简版
